# PARADOX Website Redesign & Completion
## Project Brief

**Company:** PARADOX - AI Research Lab
**Founder:** Aniket Pawar
**Target Audience:** Local Indian businesses (SMBs, startups, retailers, service providers)

### Purpose
- Personal brand + company presence
- Showcase AI products and capabilities
- Establish credibility as a serious AI lab
- Build trust with local businesses
- Position PARADOX as innovative, capable, premium, and authentic

### Brand Voice
- Modern, intelligent, warm
- Confident but not arrogant
- Technically strong but approachable
- Future-facing, original, non-template

### Quality Targets
- Lighthouse Performance >90
- Lighthouse Accessibility >90
- WCAG 2.1 AA compliant
- Responsive: Mobile, Tablet, Desktop
- Core Web Vitals: LCP <2.5s, FID <100ms, CLS <0.1
- Zero console errors
- TypeScript strict mode
- >80% component test coverage

### Tech Stack
- React 18
- Vite (build tool)
- Tailwind CSS
- React Router
- react-helmet-async (SEO)
- lucide-react (icons)

### Constraints
- Must be production-ready
- Must be accessible and inclusive
- Must load fast on Indian mobile networks
- Must feel premium, not generic
- Must showcase real AI capabilities

### Success Criteria
1. All pages complete with authentic PARADOX copy
2. All placeholder content replaced
3. SEO optimized with structured data
4. Form integration functional
5. Accessibility audit passed
6. Build produces optimized production bundle
7. Component library documented
8. Unit tests for critical components
